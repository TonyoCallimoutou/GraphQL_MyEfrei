# GraphQL_MyEfrei

Ce projet d'école consiste à mettre en place un back fonctionnel sous graphQL qui réagit de la même manière qu'un site web d'école.

## BDD
La base de données a été créé sous MYSQL, et elle comporte plusieurs tables dont :
- campus
- salles
- classes
- users
- eleves
- professeurs
- notes
- matiere
- planning
- etc...

Ainsi que des tables d'archives, et des "Trigger" pour remplir les tables d'archives.

## Dépendence
- GraphQL
- Prisma
- Dotenv
- JWT
